Hybrid logistics + multi-payment + role-based operations for Customer, Vendor, Rider, and Admin.

What is Built

Customer Ordering
Catalog discovery, cart, pincode serviceability, coupon, checkout, and order tracking timeline.
Vendor Workflow
Incoming order queue with status transitions: confirmed, preparing, ready for pickup, rejected.
Rider Workflow
Own-fleet pickup assignment, accept job, and mark out-for-delivery or delivered.
Admin Control Tower
Real-time metrics, order visibility, ticket feed, and hybrid fleet policy controls.
